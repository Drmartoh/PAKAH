# Assets Folder

This folder contains the application's logo and favicon images.

## Required Images

Please add the following images to this folder:

1. **favicon.ico** or **favicon.png** - The favicon for the browser tab
2. **logo.png** or **logo.svg** - The main PAKA HOME logo for the navbar

## Image Specifications

- **Favicon**: 32x32px or 16x16px (ICO format) or 180x180px (PNG format)
- **Logo**: Recommended size 200x60px (PNG/SVG format)

## Notes

- The logo should match the PAKA HOME brand colors (Orange #FCA311 and Blue #4A90E2)
- Images will be automatically served from `/static/assets/images/` path
